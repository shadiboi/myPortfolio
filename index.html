    Wander Log

    Online Travel Journal

    WanderLog is simply an online travel journal. With the help of Google Maps API, I was able to create a React.js app that allows users to log their journal entires while on the go. When a user posts a journal entry, WanderLog will pull the exact location that post was generated and display it on an interactive google map. Users not only have a visual map representation of all their journal logs, but also have to power to make their logs public for other WanderLog members to enjoy.

      Technologies used:
    • JavaScript
    • Node.js
    • Express.js
    • MongoDB
